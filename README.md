## Getting Started
 <img src="/src/images/preview.png" alt="Logo" width="100%">

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/YongLiam/blockchain-and-cryptocurrency.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server
   ```sh
   cd server; node index
   ```
4. Open a new terminal and run the project
   ```sh
  npm run start
   ```
